# Ryan Yan / 鄢睿洋 Personal Website

A clean one-page personal website for AI Chinese trainer applications, resume sharing, and personal branding.

## Files

- `index.html` — main website page
- `style.css` — visual design and responsive layout
- `script.js` — mobile menu and scroll reveal effect
- `assets/` — place resume PDF, profile image, or other assets here

## How to publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select `main` branch and `/root` folder.
6. Save and wait for the GitHub Pages link.

## Before publishing

- Replace the resume placeholder link in `index.html` with your actual PDF resume file.
- Optional: add a real profile photo or GitHub/LinkedIn links.
- Review the English text and adjust details if needed.
